# Parent Control

<p align="center">
  <img src="icon.png" width="150" height="150" style="border-radius: 20px;" alt="Icon" />

**ParentControl** is a simple but powerful tool that lets you monitor and manage an Android device from anywhere.  
You control everything through two easy‑to‑use interfaces:

- **Telegram Bot** – send commands and get instant updates.
- **Web Control Panel** – view history, logs, and analytics in one place.

It’s built for parents, security researchers, and anyone who needs to keep an eye on a device with permission.

---

### What You Can Do With ParentControl

Here’s a quick look at the main features:

| Category | What It Does |
| :--- | :--- |
| **Device Info** | Check if the device is online, see battery level, get GPS location, and more. |
| **Surveillance** | Take screenshots, record audio, capture photos and videos from both cameras, and set up automatic screenshots. |
| **Communications** | Read SMS messages, view call history, export contacts, and monitor clipboard content. |
| **App Control** | See installed apps, track usage, block or unblock apps, and turn blocking on or off. |
| **Security** | Lock the device, set or remove a PIN, schedule bedtime locks, and force unlock if needed. |
| **Monitoring** | Start or stop the keylogger and send captured keystrokes. |
| **Admin** | Add or remove Telegram admins, clear chat history, and set a developer contact number. |
| **Info** | Learn about the developer, find the GitHub repo, and join our community. |

> A full list of commands is available inside the Telegram bot (just type `/help`) and in the web control panel.

### Web Control Panel (Included)

All paying customers get access to a **web dashboard** that makes it easy to see everything at a glance.

You can view:

- Device status (battery, network, location).
- Command history with timestamps.
- SMS, calls, contacts, clipboard, and keylogs.
- Screenshots, audio recordings, and videos.
- GPS location history on a map.

The web panel is hosted by us – no extra setup required. You’ll get your own login details after purchase.

### System Requirements

- **Target device:** Android 9.0 or higher.
- **Control interfaces:** Telegram (on any device) + a web browser.
- **Network:** Internet connection (Wi‑Fi or mobile data).
